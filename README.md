# Ai-Robot
웹캠을 이용한 Ai로봇 컨트롤  

python과 yolov5를 이용하여 웹캠으로 객체를 인식 했을 때 로봇이 객체에 맞는 모션을 취하도록 구현된 프로그램입니다.
